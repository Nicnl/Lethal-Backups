# Lethal Backups

<img align="right" width="159px" src="vue/src/assets/lethal_backups.png">

**Lethal Backups** is a save manager for Lethal Company.  
It allows you to backup/restore your save files at wish, along with a nice interface that shows extra info (items, money, etc).
